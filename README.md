* Require
 
  * ``` python3 ```

  * ``` python3-dbus ```


* Fedora
 
  * ``` sudo dnf install python3-dbus git  ```


 
 

* To Use
 
  * ``` cd && git clone https://github.com/yucefsourani/pyfixntfs ```
 
  * ``` cd ~/pyfixntfs  ```
  
  * ``` chmod 755 pyfixntfs.py ```
  
  * ``` ./pyfixntfs.py ```
